# Clinical Trial Database System

## Overview
A SQL Server database designed to manage clinical trial studies, patient enrollment, randomization, drug kit assignment, and patient withdrawals.

## Features
- Relational database design
- Primary and Foreign Keys
- Views for reporting
- Stored Procedures for business logic
- Patient screening workflow
- Randomization workflow
- Drug kit assignment
- Withdrawal tracking

## Technologies
- SQL Server
- T-SQL
- Views
- Stored Procedures
- Transactions
