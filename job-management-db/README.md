# Job Management Database System (SQL Server)

## Overview
This project is a relational database system designed to manage job scheduling, workers, customers, vendors, materials, and job tracking for a service-based company.

## Features
- Fully normalized relational database design
- Primary and foreign key constraints
- Sample data for realistic testing
- Complex SQL queries including:
  - JOIN operations
  - Aggregations (SUM, COUNT, AVG)
  - CRUD operations (INSERT, UPDATE, DELETE)
  - Filtering and grouping logic

## Tables
- Customers
- Jobs
- Workers
- Skills
- Vendors
- Materials
- Statuses
- Linking tables for relationships

## Skills Demonstrated
- SQL Server
- Database Design (Relational Modeling)
- Data Integrity via Foreign Keys
- Query Optimization and Reporting
